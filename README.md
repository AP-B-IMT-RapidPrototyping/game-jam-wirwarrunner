# Design Doc

## Naam
WirwarRunner

## Summary
WirwarRunner is een maze game waar je doorheen een doolhof moet geraken zonder proberen geraakt te worden, hiervoor krijg je drie levens per level om het te proberen

## Genre
Maze game

## Concept Pitch
Een leuke maze game waar je onder druk van een monster dat je achtervolgt en je tot het einde moet geraken,
al dit zonder geraakt te worden.

Je krijgt drie levens om jezelf te reden van "Game Over".

## Gameplay Mechanics

### Game Flow
